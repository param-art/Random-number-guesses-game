# 🎲 Random Number Guessing Game

A simple Python-based number guessing game where the computer randomly selects a number between **1 and 50**, and the player has **8 attempts** to guess it correctly.

---

## 🚀 Features
- Computer generates a random number between 1–50.
- Player gets 8 chances to guess the number.
- Hints provided:
  - "Too high" if your guess is greater than the number.
  - "Too low" if your guess is smaller than the number.
- Displays the number of guesses left after each attempt.
- Shows the correct answer at the end of each round.
- Retry option to play again without restarting the program.
- "Game Over" message if attempts are exhausted.
